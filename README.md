⚡ Household Power Consumption Prediction
📌 Project Overview

This project aims to predict household power consumption using various machine learning regression models. It involves data preprocessing, feature selection, model training, and performance comparison.

🎯 Objectives
Predict energy consumption accurately
Compare multiple machine learning models
Analyze model performance using evaluation metrics
📊 Dataset
Dataset: Household Power Consumption
Target Variable: Global_active_power
Contains multiple features related to energy usage
⚙️ Features of the Project
Data cleaning and preprocessing
Handling missing values
Memory optimization
Feature selection (VarianceThreshold, SelectKBest)
Data scaling using StandardScaler
Model training and evaluation
Graphical comparison of models
🤖 Machine Learning Models Used
Linear Regression
Decision Tree Regressor
Random Forest Regressor
Support Vector Regressor (SVR)
XGBoost Regressor
📈 Evaluation Metrics
R² Score (Accuracy)
Mean Absolute Error (MAE)
📊 Output
Model comparison charts (R² & MAE)
Actual vs Predicted graphs
Accuracy in percentage
Sample prediction
🏆 Results
All models achieved high accuracy (>99%)
XGBoost provided the highest accuracy
Random Forest produced the lowest error
Ensemble models performed better than basic models
🚀 How to Run the Project
# Install required libraries
pip install pandas numpy matplotlib scikit-learn xgboost

# Run the script
python your_file_name.py
